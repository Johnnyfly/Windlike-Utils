# 2.1.0 (2018-09-05)

### New features
`Function` Module:
- add `debounce`
- add `throttle`

`Array` Module:
- add `shallowCompare`
- add `deepCompare`
- remove `equal`

`Object` Module:
- add `shallowCompare`
- add `deepCompare`
- remove `valueEqual`

`Math` Module:
- add `add`: add the incoming parameters together, and return the result.
  ```js
  utils.math.add(0.1, 0.2);  // 0.3
  ```
